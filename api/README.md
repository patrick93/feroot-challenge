# Feroot Challenge API

This project is API for the project [Feroot Challenge](https://github.com/patrick93/feroot-challenge)

## Requirements

- MongoDB
- NodeJS 14.x

## Project setup
```
npm install
```
### Run project
```
npm start
```

### Run project in debug mode
```
npm run debug
```

### Run your unit tests
```
npm run test:unit
```

### Lints
```
npm run lint
```
